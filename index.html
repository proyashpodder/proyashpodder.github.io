Bio
I am a PhD Candidate in the School of Computing and Information Sciences at Florida International University (FIU). Currently I am working under the supervision of Dr. Alex Afanasyev at FIU. My research interest lies in the security of computer networks, with a special focus on automation of security tools of Named Data Networking (NDN). A full list of my research publication can be found here and some high-level description of my research projects is also available here.
Before joining FIU. I was a software developer in Research and Development (R&D) department of Reve Systems Ltd. I received my Bachelors in Computer Science and Engineering from the Department of Computer Science and Engineering at Bangladesh University of Engineering and Technology in 2017.

Projects
  • SEANAC: Schema Enforced Automation of Name-based Access Control
    • Develop a policy language that allows users to describe the access control requirements of an application.
    • Construct a compiler that can generate all the necessary information based on the user policy.
    • Implement an automated system for key generation and distribution to streamline the access control process.
  • Trust Schema development for Named-Data-Networking (NDN)
    • Devise a domain-specific language that can describe trust schemas within NDN.
    • Construct a compiler capable of transforming trust-schema rules into a well-structured format, and design appropriate encoding and TLV/Binary format.
    • Develop a methodology to integrate and implement this trust schema into an NDN client library, such as ndn-cxx.
  • Hydra: Secure, Distributed Storage Framework
    • This project aims to address the challenges faced by genomics communities in terms of publication and data access by enabling secure and scalable data access for a large number of users.
    • Develop and implement the (a) security bootstrapping procedure for both Hydra nodes and users and (b) access control policies.
  • Securing Automotive Architectures with Named Data Networking
    • The objective of this project is to establish a fundamental security infrastructure for communication within vehicles.
    • Incorporate NDN to connect various link layer protocols, including CAN, LIN, and automotive Ethernet, into a cohesive and secure communication network.
  • CV2X Communication over Named Data Networking (NDN)
    • The project aims to initiate exploration of 5G CV2X/NDN technology pair by creating a comprehensive simulation package to demonstrate capabilities under various realistic network and road conditions.
    • Designed and implemented a forwarding strategy for vehicular network to build a scalable communication system by effective suppression mechanism of similar Interest and Data packets, and geographical bound.
    • Evaluated our approach in the context of a pedestrian safety application using ns-3 (ndnSIM) and SUMO as simulating platform; compare our approach with a traditional pedestrian safety application demonstrating advantages of our proposal
  • Formal Analysis of Zigbee protocol
    • Modeled various versions of Zigbee protocol and analyzed the security properties of it to discover vulnerabilities using Tamarin Prover, a security protocol verification tool.
    • Experimented with sniffed Zigbee packets to explore those vulnerabilities practically and classified different attacks.
  • Vulnerability Detection and Prevention for IoT devices
    • Developed a security mechanism for programmable smart home platforms based on user’s expectation based policy.
    • Demonstrated the generality and feasibility of our system by instantiating it for openHAB (open Home Automation Bus).
